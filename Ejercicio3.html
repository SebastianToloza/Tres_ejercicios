HTML

Lenguaje de marcado estándar utilizado para crear y estructurar páginas web. Define la estructura de contenido mediante etiquetas.

CSS

Lenguaje de hojas de estilo utilizado para describir la presentación de un documento HTML. Se encarga del diseño visual, colores, tipografías, y el layout.

JavaScript

Lenguaje de programación utilizado para agregar interactividad en las páginas web. Permite crear efectos dinámicos, validación de formularios, y aplicaciones web completas.

Python

Lenguaje de programación de alto nivel, fácil de aprender y utilizar. Es utilizado en desarrollo web, ciencia de datos, inteligencia artificial, y automatización.

Git

Sistema de control de versiones distribuido. Permite gestionar el código fuente de proyectos colaborativos y mantener un historial de cambios.

Inteligencia Artificial (IA)

Rama de la informática que busca crear máquinas capaces de realizar tareas que normalmente requieren inteligencia humana, como aprendizaje automático, reconocimiento de voz, o procesamiento de lenguaje natural

Algoritmos

Conjunto de instrucciones o pasos definidos para resolver un problema o realizar una tarea. Son fundamentales en la programación y optimización de procesos computacionales.

Blockchain

Tecnología que permite la creación de registros de información de forma segura y descentralizada, principalmente utilizada en criptomonedas como Bitcoin

Responsive Design

Técnica de diseño web que permite que una página web se adapte a diferentes tamaños de pantalla y dispositivos (como móviles, tabletas y escritorios) para ofrecer una experiencia de usuario óptima.

SEO (Search Engine Optimization)

Conjunto de prácticas utilizadas para mejorar la visibilidad de una página web en los motores de búsqueda como Google, aumentando su tráfico orgánico
